AgentSpec SDD — Phase 3.5

/validate

build → validate → ship  ·  CrewAI multi-crew architecture

A council of specialized AI crews that sits between /build and /ship. Two parallel ingestion flows (Spec + Code) feed a DeliveryCrew that generates a delta artifact, which the CouncilCrew uses to produce the final validation report, gap analysis, and production runbook.

Trigger: /validate <feature> entry point
CLI
Validate Orchestrator
Reads BUILD_REPORT, DEFINE, DESIGN, BRAINSTORM + project code tree. Spawns SpecCrew and CodeCrew in parallel.

Data Engineer
Validates pipeline spec, DAG logic, transformation logic vs DEFINE requirements.
SWE
Software Engineer

MGR

SWE
Software Engineer

ENG
Data Engineer
Reviews pipeline implementation, transformation correctness, test coverage of data logic.
OPS
DevOps Specialist
Checks CI/CD config, Dockerfile, infra code, dependency manifest, secrets hygiene.
code_report.json
DeliveryCrew — Delta Comparator new layer
CMP
Delivery Comparator
Receives original context (BRAINSTORM + DEFINE + DESIGN intents) and both crew outputs. Performs structured delta analysis: what was planned vs. what was delivered.
GAP
Gap Mapper
Maps every requirement from DEFINE to its implementation evidence. Flags missing, partial, or misaligned deliverables with severity scores.
delivery_delta.json — intent vs. delivery comparison artifact
feeds CouncilCrew
CouncilCrew — Final Verdict council
JDG
Council Judge
Reads delivery_delta.json + all reports. Computes final validation score (0–100) across 5 dimensions: alignment, quality, coverage, security, prod-readiness.
RPT
Report Writer
Generates exhaustive VALIDATION_REPORT with executive summary, per-dimension scores, gap catalog, and remediation guidance.
PRD
Production Readiness Officer
IF score ≥ threshold: generates runbook, pre-flight checklist, deployment roadmap with tasks, rollback plan, and SLA definition.
outputs

RPT
VALIDATION_REPORT_{FEATURE}.md
Full council report: score, gap catalog, findings per dimension, test results summary.
RBK
RUNBOOK_{FEATURE}.md
Step-by-step production deployment guide: pre-flight checks, deploy tasks, rollback, SLA. Only generated if score ≥ threshold.
RMP
ROADMAP_{FEATURE}.md
Gap remediation roadmap: prioritized list of missing items with effort estimates and suggested assignees.
